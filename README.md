# Job Application Tracker

## Overview

The Job Application Tracker is a simple web application that helps job seekers organize and track their job applications in one place. Users can record application details, monitor interview progress, update application statuses, and keep notes about each opportunity.

This project was built to practice front-end web development skills while creating a useful tool for managing the job search process.

## Features

- Add and manage job applications
- Track application status (Applied, Interview, Offer, Rejected)
- Record interview dates
- Store notes for each application
- Search by company name or job title
- Filter applications by status
- Edit and delete applications
- Automatically save data using browser Local Storage
- Responsive design for desktop and mobile devices

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API
- Git
- GitHub Pages

## How It Works

1. Enter the company name and job title.
2. Select the application date and current status.
3. Add interview dates and notes if needed.
4. Save the application.
5. Search, filter, update, or remove applications at any time.

All information is stored locally in the browser, so users do not need to create an account or connect to a database.
